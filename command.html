MEMEINZ COMMAND

Bot Memeinz mempunyai lebih dari 60 command yang terbagi dari 7 kategori. Kategori nya antara lain: Meme, Criminal, Fun, Minigames, Utility, Event, dan Bot. Juga beberapa command bot Memeinz antara lain: Meme, Kerang ajaib, Curi, Misi, Kuis, Gbk, Math, Event, Profile, Badge, Waifu, dan masih banyak lagi. Untuk melihat lebih lengkap gunakan command /help. Dan untuk melihat info command lebih detail, gunakan command /command {nama}.
